__Node-SplderApi2 第二版__
>基于Node 的网络爬虫 API接口 包括前端开发日报、kugou音乐、前端top框架排行、妹纸福利、搞笑视频、段子笑话、各类视频新闻资讯 热点详情接口数据

### 环境要求
>需要安装node express 

### 部署运行
```javascript
$ git clone https://github.com/ecitlm/Node-SpliderApi.git
$ npm install
### 运行
$ node app.js
```
服务器启动默认端口为3001 、启动之后就可以开启了接口服务了.
